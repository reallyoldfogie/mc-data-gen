# mc-data-gen Loader

Standalone Go module for loading and working with per-block collision data
generated by `mc-data-gen`.

## Install

```bash
go get github.com/reallyoldfogie/mc-data-gen/loader@latest
```

## Usage

```go
import (
    "fmt"
    mdl "github.com/reallyoldfogie/mc-data-gen/loader"
)

func main() {
    // Load all per-block JSON files under a version directory
    m, err := mdl.LoadBlocksDir("./data/1.21.5/blocks")
    if err != nil { panic(err) }

    // Look up a state by block ID + properties
    key := mdl.StateKey{BlockID: "minecraft:stone", PropsKey: mdl.MakePropsKey(nil)}
    info := m[key]
    fmt.Println("passable?", info.IsPassable())
}
```
